## 🚀 About The Project

This project provides a simple, user-friendly web interface to generate complex Minecraft files without needing to write JSON code by hand. Whether you're a map maker, an addon creator, or just a player who wants to customize their game, this tool is for you!

### ✨ Features

*   **Custom Recipe Creator**: Design shaped, shapeless, furnace, and other recipe types with a visual grid.
*   **JSON to .mcpack**: Automatically compiles your JSON files into a ready-to-use Minecraft behavior pack (`.mcpack`).
*   **Custom Loot Tables**: Easily define custom loot for mobs and chests, with a current focus on Armor Trims.
*   **User-Friendly Interface**: No coding knowledge required! Everything is done through an intuitive graphical interface.
*   **Instant Download**: Get your `.mcpack` file instantly, ready to be imported into Minecraft.

## 🌐 Live Demo & How to Use

Get started right away! No installation needed.

<p align="center">
  <a href="https://unknownminecraft.github.io/-WebTool-Custom-Recipe-Maker/#">
    <img src="https://img.shields.io/badge/Visit%20the%20WebTool-Click%20Here!-brightgreen?style=for-the-badge&logo=rocket" alt="Live Demo">
  </a>
</p>

1.  **Visit** the website link above.
2.  **Select** the tool you want to use (e.g., Recipe Maker).
3.  **Configure** your items, recipes, or loot tables using the on-screen options.
4.  **Click** the "Generate" or "Download" button.
5.  **Import** the downloaded `.mcpack` file into Minecraft. Enjoy!
